# OfficeBooking

Система бронирования рабочих столов.

## Стек
- Frontend: React + TypeScript + Axios
- Backend: ASP.NET Core Web API
- Database: PostgreSQL
- Auth (планируется): AD FS / OpenID Connect / JWT
- Infrastructure: Docker, Docker Compose

## Запуск проекта

```bash
cd docker
docker-compose up --build
